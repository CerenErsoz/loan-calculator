# Loan Calculator Application

This is a simple loan calculator application that allows users to calculate their monthly installments, total repayment, and payment schedule (amortization table) by entering the loan amount, annual interest rate, and term (in months).

## Features

- **Loan Calculation:** Performs detailed calculations based on the specified loan amount, interest rate, and term.
- **Amortization Table:** Provides a payment schedule showing the principal, interest, and remaining balance paid each month.
- **Modern and Professional Design:** The web interface of the application is designed to have a modern and professional look, similar to a banking application. The use of blue color has been avoided.
- **Responsive Design:** The application adapts to different screen sizes, offering a good user experience on both desktop and mobile devices.

## Setup and Running

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/loan-calculator.git
    cd loan-calculator
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the application:
    ```bash
    npm start
    ```
    The application will usually run at `http://localhost:3000`.

## Usage

1.  After starting the application, open it in your web browser.
2.  Fill in the "Loan Amount", "Annual Interest Rate (%)", and "Term (Months)" fields.
3.  Click the "Calculate" button.
4.  You will see the monthly installment amount, total repayment, and total interest, along with a detailed amortization table.

## Development Notes

The web design improvements for this project were made using Google Gemini CLI. Gemini CLI played a significant role in making the project's appearance more modern and professional, similar to a banking application.
